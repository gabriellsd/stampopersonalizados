# Logo do site

Coloque aqui a imagem da logo que aparece no menu (nav).

## Como usar

1. Salve o arquivo nesta pasta (`public/logo/`).
2. No arquivo **`src/data/site.js`**, preencha **`logoFilename`** com o nome do arquivo (ex: `logo.png`).

## Dicas

- Use **PNG** com fundo transparente ou **JPG**.
- Altura recomendada: 40 a 80 px (o site redimensiona no menu).
- Se o arquivo não existir ou der erro, o menu mostra o texto "STAMPÔ." como fallback.
